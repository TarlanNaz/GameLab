## Как играть

Карта — **шестиугольная платформа** в открытом космосе. В **центре** стоит стояка для сдачи груза, по краям — **генераторы-башни**, на которых появляются ядра.

| Шаг | Что делать |
|-----|------------|
| 1 | Вылетите с центра и найдите башню с ядром |
| 2 | Подождите полной зарядки — ядро сменит цвет |
| 3 | Подберите ядро (трюм вмещает до 5 штук) |
| 4 | Вернитесь на центральную стойку и зависните в зоне сдачи |
| 5 | Получите монеты после разгрузки |

> Незрелые ядра отталкивают корабль — подбирать можно только полностью заряженные.

## Управление

**ПК**

| | |
|---|---|
| Движение | `W` `A` `S` `D` |
| Вверх / вниз | `Пробел` / `Shift` |
| Камера | Мышь |
| Курсор | `Esc` — отпустить, клик — вернуть |

**Android**

| | |
|---|---|
| Полёт | Джойстик слева внизу |
| Высота | Кнопки **▲** и **▼** справа внизу |
| Камера | Свайп по правой части экрана |
